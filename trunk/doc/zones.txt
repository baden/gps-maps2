Работа с гео-зонами.

После создания зоны, ее будет видеть только создавший ее пользователь.
Редактировать характеристики зоны и удалять зону может только создавший ее пользователь или администратор группы.
Для зоданной зоны задается:
название, адрес, цвет, принадлежность к группе пользователей (компании).

В настройках систем, можно привязать созданную зону к какому-либо объекту.
Привязанная зона станет видимой всем пользователям, которые могут наблюдать за соответствующим объектом.
Для привязанной зоны можно указать событие:
 - тихое оповещение при покидании зоны. (сообщение будет отражено в отчетах и в событиях).
 - срочное оповещение при покидании зоны. (сообщение будет выведено на экран всем пользователям, наблюдающим за системой)
 - тихое оповещение о вхождении в зону.
 - срочное оповещение о вхождении в зону.
 - начало трека при покидании зоны (трек за сутки бует помечен маркером "Старт" при покидании зоны)
 - конец трека при вхождении в зону (трек за сутки будет помечен маркером "Финиш" при вхождении в зону).
Дополнительно может завадаться период времени действия события, только в указанный промежуток времени будет действовать выбранное условие.

Для каждой зоны может быть назначено только одно событие, но возможно повторное использование зон с другим событием.
В перспективе, создание групп правил для упрощения назначения наборов однотипных правил для большого количества объектов.
