Команды управления через SMS

Команда Описание
config Обновить конфигурацию
gps off Выключить только GPS-модуль
gps on Включить только GPS-модуль
reboot Перезагрузить устройство
power off Отключить устройство
fwupdate Обновить внутреннее программное обеспечение
gsm reboot Перезапустить GSM-модуль
iparam параметр значение Установить значение параметра
iparam gsm.server 1 Переключиться на сервер Пилигрим
iparam gsm.server 0 Переключиться на сервер Google
iparam gsm.apn 0 Настроить точку доступа для оператора Киевстар-предоплаченый.
iparam gsm.apn 1 Настроить точку доступа для оператора Киевстар-контракт.
iparam gsm.apn 7 Настроить точку доступа для оператора Utel.
iparam gsm.apn 8 Настроить точку доступа для оператора Киевстар 3G.
out on или iparam out.1 1 Активировать выход.
out off или iparam out.1 0 Деактивировать выход.
iparam out.1 8 Включить режим контроля датчика движения.
